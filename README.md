# CHM 1045 Final Exam Study Dashboard

Interactive study guide for CHM 1045 General Chemistry I, Chapters 1–12.

Built with Vite + React + Tailwind. Auto-deploys to GitHub Pages.

Live: https://michaelshapira18.github.io/chem-final-sg/

## Dev

```
npm install
npm run dev
```

## Build

```
npm run build
```

Progress (reviewed checkboxes, confidence ratings, practice-question answers, saved notes) persists via `window.storage` with a `localStorage` fallback.
